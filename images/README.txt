JAGATHI SCHOOL WEBSITE — IMAGE FILES NEEDED
============================================

Place your photos in this folder with EXACTLY these filenames:

  hero-school.jpg              → The school building exterior (wide shot — used as hero background)
  activity-cooking.jpg         → Kids in chef hats at cooking tables (hall photo)
  activity-sports.jpg          → Kids in yellow uniforms on sports ground
  activity-cooking-display.jpg → The "No Flame No Fire" cooking display close-up
  activity-event.jpg           → Kids in front of the Jagathi banner / annual event

TIPS FOR BEST RESULTS:
  • Recommended size: 1920×1080 px for hero, 800×600 px for activities
  • File format: JPG (keep under 500 KB each for fast loading)
  • You can compress images for free at: https://squoosh.app

ADDING MORE PHOTOS:
  Just add more .jpg files here and update the gallery section in index.html
  following the same pattern as the existing gallery items.

ADDING VIDEOS:
  Upload your videos to YouTube, then replace the video-placeholder divs in index.html with:
    <div class="video-embed">
      <iframe width="100%" height="360"
        src="https://www.youtube.com/embed/YOUR_VIDEO_ID"
        frameborder="0" allowfullscreen></iframe>
    </div>
